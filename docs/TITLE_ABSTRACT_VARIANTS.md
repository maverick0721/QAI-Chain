# Title and Abstract Variants

## Title Variant A (Systems + Reproducibility)

QAI-Chain: Reproducible Co-Design of Quantum, AI, and Blockchain Components with Contract-Validated Interfaces

## Title Variant B (Security + Governance)

QAI-Chain: A Hybrid Framework for Post-Quantum Secure and AI-Guided Decentralized Governance

## Title Variant C (Engineering + Evaluation)

QAI-Chain: An Artifact-Driven Research Stack for Quantum-AI-Blockchain Integration

## Abstract Variant A (Conservative Systems Framing)

We present QAI-Chain, a modular research prototype that combines blockchain networking, post-quantum transaction security interfaces, reinforcement learning based governance, and quantum model components. We focus on engineering rigor and reproducibility rather than broad performance claims: typed API contracts, integration and contract testing, static checks, and executable artifact generation are first-class design goals. On a multi-seed governance simulation, heuristic control outperforms policy optimization in the present simplified environment, while PPO provides partial adaptation over random behavior. We report confidence intervals and permutation tests, and provide a pipeline that regenerates manuscript tables from experiment artifacts. QAI-Chain is positioned as a research engineering substrate for future hybrid-system investigation.

## Abstract Variant B (Method-Centric Framing)

QAI-Chain is a unified stack for studying interactions between decentralized protocols, post-quantum cryptographic interfaces, and hybrid AI/quantum components. We introduce an artifact-driven workflow in which experiment results are persisted as machine-readable outputs and manuscript tables are auto-generated from those outputs. We evaluate random, heuristic, and PPO-based governance controllers under an expanded multi-seed protocol and include ablations plus statistical significance testing. Findings highlight that strong simple baselines remain competitive in constrained environments, motivating richer dynamics for future learning advantages. The released framework emphasizes transparent, reproducible comparison over one-off demonstration.

## Abstract Variant C (Security/Robustness Emphasis)

Hybrid decentralized intelligence systems require both adaptive behavior and robust interfaces. We present QAI-Chain, a prototype integrating blockchain and P2P primitives, contract-validated RPC APIs, post-quantum signature pathways, and AI/quantum model components. We pair architectural integration with software quality gates and reproducible evaluation scripts, then study governance control performance across baselines and PPO variants. While heuristic control is strongest in the current simplified environment, the framework exposes where learned policies improve and where they fail, with statistical reporting and explicit limitations. The resulting repository provides a security-aware and reproducibility-focused foundation for subsequent large-scale hybrid experiments.
