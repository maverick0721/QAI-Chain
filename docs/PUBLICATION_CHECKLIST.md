# Publication Checklist

## Reproducibility

- [x] Experiment scripts are executable from repository root
- [x] Results artifacts are generated as JSON and markdown
- [x] Manuscript tables are generated from artifacts
- [x] CI quality gates (lint, type checks, tests, healthcheck)

## Empirical Reporting

- [x] Baseline comparisons (random, heuristic, PPO)
- [x] Ablation analysis for PPO variants
- [x] Multi-seed protocol for publication run
- [x] Confidence intervals and significance tests

## Paper Quality

- [x] Abstract with bounded claims
- [x] Explicit limitations section
- [x] Threat model and security considerations
- [x] Ethics and broader impacts section

## Venue Preparation

- [x] Format manuscript to target template (e.g., NeurIPS, ICLR, IEEE)
- [x] Ensure bibliography style matches venue requirement
- [x] Add author metadata and affiliations
- [x] Produce camera-ready PDF and source bundle
- [x] Final proofreading pass for grammar and consistency

## Submission Bundle

- [x] PDF compiled and validated
- [x] Source tarball prepared
- [x] Artifact links verified in README
- [x] License and citation block included
